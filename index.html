Spring

The Spring Framework is an open source application framework and inversion of control container for the Java platform. The framework's core features can be used by any Java application, but there are extensions for building web applications on top of the Java EE (Enterprise Edition) platform. Although the framework does not impose any specific programming model, but helps a lot with development, it has become popular in the community as an addition to, or even replacement for the Enterprise JavaBeans (EJB) model.
